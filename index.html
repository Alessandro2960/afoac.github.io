Durante a segunda guerra mundial estava sendo construído um computador para melhorar os acertos balísticos, também conhecido como ENIAC. Este projeto era financiando exclusivamente pelo exército dos EUA e era composto dois cientistas, John Eckert e John Mauchly. Depois disto, apesar dos recursos financeiros empregados, o projeto foi concluído com sucesso, mas depois do fim da segunda guerra mundial o que tornou-o inutilizável para tal fim, pois o seu objetivo de construção havia terminado e consequentemente o seu propósito foi encerrado. Apesar das circunstâncias, o projeto do computador ENIAC iria revolucionar a tecnologia existente e definir a base da arquitetura para a computação através do modelo de Von Neumann ao qual iremos retratar detalhadamente ao longo da discussão.

Até o presente momento, os computadores eram construídos para tarefas específicas(computadores de programa fixo), assim caso a necessidade do projeto passasse por alguma alteração ou os objetivos definidos inicialmente fossem alterados, consequente o projeto de construção do computador teria que ser totalmente remodelado para as novas necessidades. Pensando em tal problemática, os mesmos cientistas iniciaram um novo projeto chamado de EDVAC, mas com um novo integrante matemático chamado John Von Neumann que escreveu um artigo sobre o EDVAC que estavam inclusos seus componentes, estruturas lógicas e matemáticas definindo assim um novo modelo de arquitetura que é utilizado pela maioria dos computadores atuais chamado, Arquitetura Von Neumann.

John Von Neumann seguiu uma linha de raciocínio de lógica e construção totalmente diferente do que existia até aquele momento, as principais características são: Construção de um computador de propósito geral e que fosse alterado facilmente de acordo com os objetivos definidos, armazenamento de dados e instruções em um único local, unidade de processamento, unidade lógica,unidade aritmética e unidade de controle. Isso permitiu que atualmente, o mesmo computador que você utiliza para entrar na internet, também possa ser utilizado para realização de cálculos, jogos, processamento de imagens e infinitas soluções, abaixo temos um diagrama de como é a organização dessa arquitetura.
